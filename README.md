1. Install `hexo-cli` through `npm`:

    ```sh
    npm install hexo-cli -g
    ```

2. Clone the repository and `cd` into it:

    ```sh
    git clone https://github.com/jktrn/blog.git
    ```

    ```sh
    npm install
    ```
